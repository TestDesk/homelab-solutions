# Homelab Solutions

A wiki for self-hosted solutions, fixes, useful extensions & custom builds. Real problems, tested solutions — no guesswork.

📖 [Browse the Wiki](https://github.com/testdesk/homelab-solutions/wiki) — all solutions, fixes and guides in one place.

## Structure

Documentation is organized by platform and topic in a directory structure. Some pages are available in both English and German.

You can either browse the directories directly or use the wiki, which links to all documents.

```
yunohost/
└── apps/
    └── cryptpad/
        ├── screenshots/
        ├── yh-cryptpad-sandbox-nginx.md
        └── yh-cryptpad-sandbox-nginx_de.md
```
